# Python虚拟环境设置说明

## 概述

本项目使用PyCaret库进行AutoML建模，需要Python 3.10或3.11版本支持。为避免与系统Python环境冲突，我们创建了虚拟环境配置脚本，方便您快速搭建独立的项目运行环境。

## 环境要求

- Python 3.10或3.11（推荐3.10，兼容性最佳）
- 约2GB磁盘空间（用于安装依赖库）
- 建议4GB以上内存（PyCaret运行时需要）

## 快速设置

1. **设置虚拟环境**：
   - 双击运行`setup_env.bat`脚本
   - 脚本会自动创建名为`venv`的虚拟环境并安装所有依赖
   - 首次安装可能需要5-10分钟，取决于网络速度

2. **运行项目**：
   - 设置完成后，可以直接双击`run_project.bat`运行项目
   - 或者手动激活环境后运行：
     ```
     venv\Scripts\activate.bat
     python main.py
     ```

3. **退出环境**：
   - 在命令行中输入`deactivate`即可退出虚拟环境

## 依赖说明

主要依赖库包括：
- PyCaret 3.0.4（AutoML框架）
- pandas 1.5.3（数据处理）
- scikit-learn 1.2.2（机器学习算法）
- matplotlib 3.7.1 和 seaborn 0.12.2（可视化）
- xgboost、lightgbm、catboost（高性能模型）

完整依赖列表请查看`requirements.txt`文件。

## 常见问题

1. **安装依赖失败**
   - 检查网络连接
   - 尝试使用国内镜像：编辑`setup_env.bat`，将pip安装命令改为：
     ```
     pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
     ```

2. **PyCaret导入错误**
   - 确认已正确激活虚拟环境
   - 检查Python版本是否为3.10或3.11
   - 尝试重新安装：`pip install pycaret==3.0.4`

3. **内存不足错误**
   - 减少数据集大小或关闭其他内存占用程序
   - 在`src/AutoML.py`中修改PyCaret设置，减少模型数量或交叉验证折数

## 注意事项

- 首次运行PyCaret时，它会自动下载一些额外资源，可能需要一些时间
- 建议在虚拟环境中运行所有与项目相关的代码，以避免依赖冲突
- 如需添加新的依赖，请先激活环境，然后使用`pip install`安装，并更新`requirements.txt`
